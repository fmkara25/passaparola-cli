# Passaparola CLI (Python)

Basit bir terminal (CLI) Passaparola oyunu.
Python öğrenenler için adım adım geliştirilmiştir.

## Oyun Kuralları
- Şimdilik A, B, C harfleri vardır
- Her harf için bir soru bulunur
- Cevaplar:
  - Doğru → DOGRU
  - Yanlış → YANLIS
  - `pass` → PAS (sonra tekrar sorulur)
- PAS kalan soru kalmayınca oyun biter

## Çalıştırma
```bash
python main.py
